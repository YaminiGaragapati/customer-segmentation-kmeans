# 📊 Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project develops an end-to-end Customer Segmentation System using Python and K-Means Clustering.  
The system categorizes customers based on purchasing behavior and demographic characteristics to help businesses improve marketing strategies and customer engagement.

---

## 🎯 Objective
To identify distinct customer groups that enable businesses to:
- Target marketing strategies
- Personalize recommendations
- Optimize customer engagement
- Improve revenue generation

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

---

## 🔍 Methodology

### 1️⃣ Data Cleaning & Preprocessing
- Loaded dataset
- Handled missing values
- Verified data consistency

### 2️⃣ Feature Engineering
- Created Age feature
- Created Total Spending feature

### 3️⃣ Data Scaling
- Applied StandardScaler to normalize features

### 4️⃣ Optimal Cluster Selection
- Used Elbow Method to determine optimal number of clusters

### 5️⃣ K-Means Clustering
- Segmented customers into 3 clusters

### 6️⃣ Visualization & Analysis
- Scatter plot for cluster visualization
- Centroid plotting
- Spending distribution boxplot

---

## 📈 Key Insights

### 💎 Premium Customers
High income and high spending customers contributing significantly to revenue.

### 🎯 Potential Customers
Moderate income customers with growth opportunity.

### 📉 Budget Customers
Low income and low spending customers.

---

## 🚀 Business Impact
This segmentation approach allows businesses to:
- Design targeted marketing campaigns
- Improve personalization strategies
- Enhance customer retention
- Optimize revenue growth
